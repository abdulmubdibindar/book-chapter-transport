# Dokumentasi Quarto

Direktori ini berisi dokumentasi tentang penggunaan Quarto sebagai alat produksi dokumen serbabentuk tulisan seperti buku, laporan, artikel ilmiah.

Susunan direktori ini terdiri atas

- [`@01-authoring/`](00-01-overview.md) : berisi petunjuk untuk benar-benar menuliskan apa yang mau kita tuliskan
- [`@02-computations/`](00-02-overview.md) : berisi petunjuk hal-hal yang berkenaan dengan melaksanaan komputasi dengan bahasa pemrograman R dan Python
- `@03-tools/` : berisi petunjuk bagaimana mengimplementasikan penggunaan Quarto dalam VS Code
