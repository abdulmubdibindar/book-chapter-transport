# Dokumentasi Quarto

Direktori ini berisi dokumentasi tentang penggunaan Quarto sebagai alat produksi dokumen serbabentuk tulisan seperti buku, laporan, artikel ilmiah.

Susunan direktori ini terdiri atas:

- [`@01-authoring/`](01-authoring/00-01-overview.md) : berisi petunjuk untuk benar-benar menuliskan apa yang mau kita tuliskan
- [`@02-computations/`](02-computations/00-02-overview.md) : berisi petunjuk hal-hal yang berkenaan dengan melaksanaan komputasi dengan bahasa pemrograman R dan Python
- [`@03-tools/`](03-tools/00-03-overview.md) : berisi petunjuk bagaimana mengimplementasikan penggunaan editor Quarto seperti VS Code
- [`@04-documents/`](04-documents/00-04-overview.md) : berisi petunjuk tentang pembuatan format dokumen PDF dan Word
- [`@07-websites/`](07-websites/00-07-overview.md) : berisi petunjuk untuk membuat website dan blog menggunakan Quarto
- [`@08-books/`](08-books/00-08-overview.md) : berisi petunjuk khusus mengenai penyusunan buku (e-book, majemuk HTML, PDF) dan referensi silangnya
