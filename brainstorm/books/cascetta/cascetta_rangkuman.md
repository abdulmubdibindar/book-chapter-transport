# Rangkuman Buku Cascetta

- [`@@cascetta2009transportation.md`](@cascetta2009transportation.md) : ikhtisar komprehensif buku teks Transportation Systems Analysis karya Ennio Cascetta mengenai rekayasa dan analisis sistem transportasi
- [`@cascetta_chp_1.md`](cascetta_chp_1.md) : eksplorasi fondasi teoretis sistem transportasi dan memodelkan interaksi permintaan dengan aspek perilaku manusia
- [`@cascetta_chp_2.md`](cascetta_chp_2.md) : model matematis penawaran transportasi untuk mensimulasikan kinerja infrastruktur, aliran jaringan, dan lalu lintas
- [`@cascetta_chp_3.md`](cascetta_chp_3.md) : penjabaran teori utilitas acak (Random Utility Theory) untuk memodelkan pilihan diskrit terkait perilaku transportasi
- [`@cascetta_chp_4.md`](cascetta_chp_4.md) : panduan komprehensif terkait pemodelan permintaan perjalanan sebagai turunan dari kebutuhan sosial ekonomi
- [`@cascetta_chp_5.md`](cascetta_chp_5.md) : pemodelan penugasan lalu lintas statis dasar (Basic Static Assignment Models) untuk simulasi interaksi permintaan dan penawaran
- [`@cascetta_chp_6.md`](cascetta_chp_6.md) : tinjauan model pembebanan lalu lintas tingkat lanjut seperti pembebanan multikelas dan dinamika antarkeseimbangan
- [`@cascetta_chp_7.md`](cascetta_chp_7.md) : penjelasan mengenai pemodelan dinamis dalam-hari (within-day dynamics) terhadap transisi aliran lalu lintas dan antrean waktu nyata
- [`@cascetta_chp_8.md`](cascetta_chp_8.md) : panduan teknik empiris dan statistik untuk mengestimasi dan memprediksi aliran permintaan perjalanan transportasi
- [`@cascetta_chp_9.md`](cascetta_chp_9.md) : pembahasan model perancangan dan optimasi penawaran transportasi dengan mempertimbangkan reaksi pengguna jaringan
- [`@cascetta_chp_10.md`](cascetta_chp_10.md) : kerangka kerja evaluasi kelayakan proyek tunggal serta perbandingan berbagai alternatif intervensi sistem transportasi
